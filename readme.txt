== Description ==

The Post Institute Affiliate Plugin is designed to help affiliates earn money by introducing new customers to Post Institute.Reactive Attachment Disorder, Oppositional-Defiant Disorder, Post-Traumatic Stress Disorder, Bi-Polar, Depression, Attention Deficit Hyperactivity Disorder are all behaviors that bring parents to The Post Institute. Both parents and professionals alike who are parenting and working with adopted as well as foster, and biological children come to the Post Institute seeking hope. Use this plugin to easily show product details from Post Institute on any post or page within your Wordpress site. When customers follow links from those product and make a purchase at Post Institute, you will earn highly competitive commissions per successful referral. 

You must be a member of the Post Institute Affiliate program.

= Key features =
* Show Post Institute product image and details on any Wordpress post or page using a simple text string (shortcode).
* Show a carousel of Post Institute products in a sidebar widget.

* Direct buy option at the results page with your affiliate link.

== Installation ==

You can install the plugin automatically from within your WordPress interface. Simply log in to WordPress, select Plugins from the menu and then Add New. Search for "Post Institute Affiliate" and click the install link to automatically install the plugin.

For older versions of WordPress or manual installation: 1. Download and unzip the plugin zip file. 2. Upload the entire folder to the /wp-content/plugins/ directory of your WordPress installation.

Once you have the plugin installed, activate it through the in WordPress 'Plugins' menu. Look for the Post Institute Affiliate link at the bottom of the left-hand column of your WordPress admin site. Visit the Post Institute panel within your WordPress admin site to enter your affiliate code, and to adjust your options.
